# nanoctl -- configure your Korg Nanokontrol 2.

The `nanoctl` command lets you view and set the configuration of your [Korg
nanoKontrol2]. You can change the midi control code assigned to different
controls, change buttons from momentary to toggle mode, control the various
LEDs on the board, etc.

[korg nanokontrol2]: https://www.korg.com/us/products/computergear/nanokontrol2/

You can set the configuration using either command line arguments or a JSON file.
